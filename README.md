# ChromeXt

This repository is a personal fork of [JingMatrix/ChromeXt](https://github.com/JingMatrix/ChromeXt).

ChromeXt adds UserScript and DevTools support to Chromium-based and WebView-based Android browsers through the Xposed framework. For the original project documentation, usage details, and API notes, see [README.upstream.md](README.upstream.md).

## About This Fork

This fork keeps the upstream ChromeXt behavior and adds practical support and usability improvements focused on Xiaomi Browser / MIUI Browser.

Main changes in this fork:

- Added Xiaomi Browser package support for `com.android.browser` and `com.mi.globalbrowser`.
- Added a Xiaomi-specific WebView hook path for MIUI / HyperOS browser WebView classes.
- Improved injection timing for Xiaomi Browser cold starts, page restore, history restore, WebView attach, and Activity resume paths.
- Added duplicate-injection guards so recovery probes do not repeatedly register ChromeXt or destabilize the browser.
- Added a local script manager page served at `https://chromext.local/`.
- Added an LSPosed module settings entry that opens the local script manager in the target browser.
- Redesigned the script manager UI for mobile editing:
  - script list and detail-first layout
  - separate Match / Include / Exclude editors
  - source view opened only on demand
  - long names clipped cleanly
- Modernized the `.user.js` install prompt, including reinstall detection and install feedback.
- Removed heavyweight source highlighting from the manager for better stability on mobile.

## Build

Debug build:

```bash
./gradlew assembleDebug
```

Release build:

```bash
./gradlew assembleRelease
```

The release APK generated by Gradle is unsigned unless a signing config is added locally.

## Notes

This fork is experimental and currently tuned around Xiaomi Browser behavior observed on a MIUI / HyperOS device. Other Chromium/WebView browsers should still follow upstream behavior unless specifically touched by the Xiaomi compatibility path.

Upstream project: [JingMatrix/ChromeXt](https://github.com/JingMatrix/ChromeXt)
